# starmer
online sneed act (formerly chuck's)

## requirements
- [tor expert bundle](https://www.torproject.org/download/tor/), but this should work with any socks5 proxy

## usage
1. run tor.exe
2. run starmer.exe
3. control panel -> internet options -> connections -> lan settings -> [x] use a proxy server -> advanced -> socks: localhost:1337
4. browse the web
